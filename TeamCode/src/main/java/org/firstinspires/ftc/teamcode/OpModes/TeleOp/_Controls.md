# 24579 Robot Code

## Controls!
### Gamepad1:
 -  Right Stick
    - Move Forward 
    - Strafe
 - Left Stick
   - Rotate 
 - Left Bumper 
   - Switch to Field Based Movement 
 - A 
   - Reset IMU Sensor for Field Based Movement 
### Gamepad2:
 - Right Trigger 
   - Rotates Intake Forward 
 - Left Trigger 
   - Rotates Intake Reverse

## Light Controller information

 - Red = 0.278
 - Orange = 0.3081
 - Yellow = 0.3502
 - Green = 0.5
 - Teal = 0.52
 - Blue = 0.61
 - Purple 0.69
 - Pink = 0.7205
 - White = 1
